# Tout.Social Buttons #

Adds social sharing buttons to your posts.



## Description ##

Adds social sharing buttons to your posts.



## Installation ##

1. Upload `tout-social-buttons` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
